# Issue #48 — Tuple constructor helper

目的:
- 標準ライブラリにタプル(Tuple)を簡単に構築・操作するためのヘルパー関数を追加する提案。

提案 API（例）:
- `~tuple2 ~a ~b` : 2要素タプルの構築（例: `(a,b)` の構文糖）
- `~fst ~t` : 2要素タプルの1要素目を取得
- `~snd ~t` : 2要素タプルの2要素目を取得

設計ノート:
- LZSCR の既存のデータ構造と整合するよう最小限の API を提供。
- まずはドキュメント + 使用例を追加し、後続で実装（stdlib への実装ファイル追加）を行う。

使用例:
```lzscr
~pair = (~tuple2 1 2);
~a = (~fst ~pair); # => 1
~b = (~snd ~pair); # => 2
```
