# 構文（現状）

- リテラル:
  - Unit: `()`
  - Int: `42`
  - Str: `"hi"`（\エスケープ可）
- 識別子:
  - 参照: `~name`
  - 裸シンボル: `Name`（値として保持、将来のコンストラクタ変数候補）
  - メンバ風シンボル: `.println` → シンボルとしてパース
- ラムダ: `\x -> expr`
- 適用: `(f x)`（左結合）
- ブロック: `{ expr }`
- 糖衣構文:
  - `!sym` → `(~effects .sym)`
- 予約/トークン: `~`, `!`, `.ident`, `(`, `)`, `{`, `}`, `,`, `->`, `\\`

例:
```
(~add 1 2)
(~to_str 42)
!println "ok"
(~seq () (!println "ok"))
(\x -> x) 10
```
